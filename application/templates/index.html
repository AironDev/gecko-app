The project

GECko+ is built on top of two Artificial Intelligence models. To correct spelling and grammar mistakes, we use GECToR, a Grammatical Error Corrector model developed by the well-known Grammarly. For tackling discourse fallacies, we make use of a sentence ordering model based on Google's BERT.

We have created this web-app in the framework of the Software Project for the second year of our MSc in Natural Language Processing at the Université de Lorraine. The source code is available on GitHub.
