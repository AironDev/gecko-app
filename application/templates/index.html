GECko is a Grammar Error Corrector for English. It is able to correct various spelling and grammatical mistakes. Its functioning is simple and quick: just try to type an English sentence (with some mistakes, of course!) on the left box. Our GECko, which is always hungry, will wake up and gobble up all the mistakes you have made!

GECko is built on top of GECToR, a Grammar Corrector model developed by the well-known Grammarly. We have created this web-app in the framework of the Software Project for our Master of Science's second year. The source code is available on GitHub.
